Selects

Six select-control compositions modelled on Flowbite's reference. Triggers are buttons with explicit icon DOM elements so Figma captures both the value text and chevron correctly.

1 · Default select

Single-select trigger with chevron on the right. Selected value shown as text.

2 · Sizes

Three sizes — small (32px), default (40px), large (48px). Match the Input size scale.

3 · Multiple select

Scrollable list — selected items highlighted with brand background. Hold ⌘ / Ctrl to add to the selection.

4 · Disabled

Greyed-out, no interaction. Communicates "value locked" — used when a select is contextually fixed.

5 · Underline style

Minimal — bottom border only, transparent background. For tight inline forms or table-cell selects.

6 · Country picker

Custom flag trigger joined to a region select — one segmented control.
