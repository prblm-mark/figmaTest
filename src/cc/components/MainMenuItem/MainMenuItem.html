MainMenuItem

The atomic row used inside the Menu panel. 4 component states from Figma — Default, Hover, Selected, Expanded. Rows always fill their parent container's width.

State = Default

Transparent bg. Label uses --ai-text-invert-secondary; icon uses --ai-icon-invert-secondary (the dusty pair).

State = Hover

Pinned via inline style so the static showcase captures it. In app, this kicks in via :hover. Bg goes to --cc-mainmenu-primary-bg; label and icon colours stay at the Default duotone. Items with a submenu carry a trailing chevron-down.

State = Selected

Bg --cc-mainmenu-primary-bg. Label brightens to --ai-text-invert and icon to --cc-mainmenu-icon. No chevron rotation — this is a terminal item, not a submenu trigger.

State = Expanded

Same colour treatment as Selected, plus the chevron is flipped (scaleY(-1)) so chevron-down points up — signalling the submenu below is open.
