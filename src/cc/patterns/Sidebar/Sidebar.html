Narrow icon rail for CC. 5 variants — Desktop × {Default, Selected} and Mobile × {Default, Selected, Expanded}. Pairs with the Menu panel inside the SidebarMenu composite.

Device = Desktop

Desktop Default and Selected are visually identical — the workspace button (SlidersVertical) is highlighted in both. Brand cell is 40×72 (taller slot fits the wordmark mark).

Device = Mobile

Mobile rail is 4 px narrower (52 vs 56) with tighter gap. Brand cell shrinks to 40×40. Default has no active highlight; Selected highlights SlidersVertical; Expanded adds 5 secondary tool buttons after the EllipsisVertical separator.
