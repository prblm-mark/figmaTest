AutoAnswers

AI auto-answer card for support forums. Six states across the answer lifecycle — Processing → Suggested → (Complete / Partial / Inadequate), plus a No Answer fallback. Theme-agnostic --ai-* tokens; rendered here under data-brand="cc". Responsive via container queries — each card adapts to its container width (MD breakpoint = 768px), not the viewport.

All variants — container-responsive

Every variant below shares one resizable container. Drag the frame's right edge or use the presets to move the whole set across the MD breakpoint (768px): below it the header stacks, type shrinks, and buttons drop to the small size and stack. The SuggestedAnswer card is live — its three feedback buttons resolve it to a terminal marked state; use Reset to replay.
