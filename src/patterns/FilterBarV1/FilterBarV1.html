Filter Bar — V1

Single-row filter toolbar (A/B variant 1). Composes the saved-views Dropdown, FilterItem chips (V1 uses the dashed “empty” chip), search Input, and a kebab actions menu. Resize below 768px to see the mobile stack (views + actions on top, chips wrap below).

Default — interactive

Open “All Orders”, toggle a chip, hit 🔍 to enter search, ⋮ for actions, or “+ New view” to rename.
