Can you explain page views, and how they are saved in Affino?

Page views refer to the total number of times pages on your site are viewed by users. In Affino, page views are counted each time a page loads and are tracked as part of your site analytics. Multiple views of the same page by the same user are counted separately.

Key points about page views in Affino

  • Page views are recorded under the Stats function within each article or page.
  • They include views by both registered and guest users.
  • If a user clicks on an article but encounters a paywall, it still counts as a page view.
  • Page views are used in various reports and dashboards, such as the Analysis Dashboard, to provide insights into user engagement.
  • Affino distinguishes between simple page views (impressions) and content consumption, where consumption indicates the user has actively engaged with the content beyond just loading the page.

You can view page view statistics in Affino via:

  • Control > Publish > Articles + Stats
  • Control > Analyse > Articles Report
  • The Analysis Dashboard for broader site metrics including page views over time

This tracking helps you understand how often your content is accessed and supports optimizing engagement and conversion strategies.
