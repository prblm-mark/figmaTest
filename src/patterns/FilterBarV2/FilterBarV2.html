Filter Bar — V2

Two-row filter toolbar (A/B variant 2). Row 1: saved-views + an explicit Export button + search/kebab. Row 2: the filter chips (V2 uses the solid FilterItem; only “Add Filters” is dashed). Resize below 768px — the Export button collapses out of the bar (per Figma).

Default — interactive

Open “All Orders”, toggle a chip, hit 🔍 to search, ⋮ for actions, or “+ New view” to rename.
