SidebarMenu

The Control Centre app chrome — Sidebar rail + Menu panel side-by-side. Live composite: rail buttons swap the panel Type; MainMenuItem rows expand inline; search filters live; CRM recent has a show more/less toggle; the items list scrolls and hides the brand + search when it does. Mobile composite adds an Ellipsis-toggled secondary tool rail.

Device = Mobile

Mobile rail (52) + Menu panel (280). Default state is rail-only — no menu open, nothing selected. SlidersVertical opens Control, UserCog opens CRM, ChartNoAxesCombined opens Analysis. Star / Network / CircleUser are buttons without a menu. The Ellipsis reveals 5 extra tool buttons; tap again to collapse.
