Search Inputs

Six search-input compositions modelled on Flowbite's reference, built using Affino Design System tokens and existing Input + Button components.

1 · Basic search · button overlaid

Standard search bar — leading icon, primary CTA tucked inside the trailing edge of the field.

2 · Search with category dropdown

Segmented control — category selector on the left, joined input, search button on the right.

3 · Split · input and button separate

Input and submit are visually separate — gap between them, both pick up the primary focus ring on interaction.

4 · Location search

Region/flag selector + address input + submit. Useful for postcode lookups or geo-scoped queries.

5 · Voice search

Leading magnifier, optional mic for voice input, primary submit. Mic acts as a tertiary action — same height as the field.

6 · Advanced · two flanking dropdowns

Pre-/post-input dropdowns — protocol on the left, TLD on the right. Useful for URL- or schema-style inputs.
